# FSociety Dark System

FSociety Dark System is a single‑file, browser‑based mini operating system designed with a dark hacker aesthetic. It includes login, registration, a file manager, text editor, and a terminal with custom commands. Everything runs locally using `localStorage`, so data persists without any backend.

## Features

- **Login & Registration**  
  Built‑in popup registration system with persistent user accounts.

- **User Environment**  
  Each user has their own files and settings.  
  The CEO account has full control and access to global commands.

- **File Manager**  
  Create, edit, and delete files with permanent storage.

- **Text Editor**  
  Simple integrated editor for notes, scripts, or anything else.

- **Terminal Emulator**  
  Includes basic commands like `help`, `ls`, `cat`, `whoami`, and `clear`.  
  The CEO account can run global commands affecting all users.

- **Dynamic Dark Themes**  
  The interface theme changes based on the logged‑in user.

- **Boot Screen**  
  Custom startup animation with a user‑defined image.

- **Single‑File Application**  
  Everything is contained in one HTML file. No dependencies, no server.

## How to Use

1. Download `fsociety.html`
2. Open it in any modern browser
3. Register or log in
4. Explore the system

## GitHub Pages

To publish it as a website:

1. Upload `fsociety.html` to a GitHub repository  
2. Go to **Settings → Pages**  
3. Set:
   - Branch: `main`
   - Folder: `/root`
4. GitHub will generate a public link you can share

## Purpose

This project is meant to be a fun, stylish, and educational experiment that shows how far a single HTML file can go when combining UI, storage, and interactivity.
